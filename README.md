Donum Cashier App adalah aplikasi kasir sederhana yang digunakan untuk mengelola transaksi penjualan Donat Kentang Numnum. Aplikasi ini menyediakan fitur dashboard untuk melihat total pemasukan, jumlah transaksi, dan jumlah pesanan secara cepat. Selain itu, terdapat fitur riwayat penjualan yang membantu pengguna memantau dan mencatat setiap transaksi yang telah dilakukan sehingga pengelolaan penjualan menjadi lebih rapi dan efisien.![upscalemedia-transformed](https://github.com/user-attachments/assets/cddd6ecf-6150-4867-be98-b9a717a991ab)
