# 🍩 Donum Cashier App

Aplikasi **Donum Cashier App** merupakan sistem kasir (Point of Sale) yang digunakan untuk membantu pengelolaan transaksi penjualan pada usaha **Donat Kentang Numnum**.  

Aplikasi ini dirancang untuk mempermudah proses transaksi, pencatatan pemasukan, serta pengelolaan stok barang secara lebih **cepat, rapi, dan terorganisir**.

Selain itu, sistem ini juga memanfaatkan **RFID Card** sebagai metode login kasir sehingga proses autentikasi menjadi lebih praktis dan modern.

Project ini dikembangkan sebagai bagian dari **pembelajaran pemrograman Java dan pengembangan aplikasi desktop berbasis sistem informasi kasir**.

---

# 🚀 Features

Beberapa fitur utama dalam aplikasi ini antara lain:

### 🔐 RFID Login
Sistem login kasir menggunakan kartu RFID sehingga pengguna dapat masuk ke sistem dengan cepat tanpa perlu mengetik username dan password.

### 💳 Transaksi Penjualan
Kasir dapat melakukan proses transaksi penjualan produk secara langsung melalui sistem.

### 🧾 Cetak Struk
Setiap transaksi yang berhasil akan menghasilkan **struk pembayaran** yang dapat dicetak sebagai bukti transaksi pelanggan.

### 📦 Manajemen Stok Barang
Sistem dapat mencatat dan mengelola stok barang sehingga jumlah stok selalu terpantau.

### 🛒 Pemesanan
Fitur untuk mencatat pesanan pelanggan sebelum diproses menjadi transaksi penjualan.

### 💰 Pengeluaran
Mencatat pengeluaran operasional usaha sehingga keuangan dapat dipantau dengan lebih baik.

### 📊 Dashboard Penjualan
Menampilkan ringkasan informasi seperti:
- Total pemasukan
- Jumlah transaksi
- Jumlah pesanan
- Riwayat penjualan
